1.jQueryの使い方
    ・直接インストール
    →
        公式ページからダウンロードし、プロジェクトに直接配置する方法。
        容量使ったりするのであまり使わない
        オフライン環境で使う場合はこっち
    ・CDNを使用する方法
    →
        公式ページやjsDelivrを使って
        HTMLにタグを埋め込んで使用する方法
        よく使用する。
        オンライン環境が必須。

2.jQueryの構文
    基本的な構文は
        $(selector).action()
        
        ・$サインはjQueryを定義/アクセスするためのもの。
        ・(selector)は、HTML要素のクエリ(または検索)のもの。
        ・action()は要素上で実行するためのもの。

        例
            ・$(this).hide() - 現在選択している要素を非表示にする
            ・$("p").hide() - すべての<p>要素を非表示にする。
            ・$(".test").hide() - class = "test" であるすべての要素を非表示にする
            ・$("#test").hide() - id = "test" であるすべての要素を非表示にする

3.Document Readyイベント
    文書の読み込みが完了(して準備ができる)前にjQueryコードの実行を防ぐために使用する。

    これは、文書の<head>内部にJavaScriptを読み込む記述を可能にするためにする。
    この対処方をしなかった場合、
    ・まだ存在しない要素を非表示にしようとする
    ・まだロードしていない画像のサイズを取得しようとする
    などといった部分でエラーが出る。
    このようなエラーを防ぐために基本的にDocument Readyイベントを使用して、
    この内部にjQueryのコードを記述していく。

    $(function(){
        //  ここにjQueryのコードを記述する  //
    });

4.jQueryセレクタ
    jQueryセレクタは、HTML要素の選択と操作を可能にする。

    HTML要素のid/class/type/属性/属性値に基づいて、
    HTML要素を"検索"(または選択)するために使用する

5.elementセレクタ
    jQueryのelementセレクタは、要素名に基づいてHTML要素を選択する
    
    構文
    $("test")

    例
    chap1-1-element.html/chap1-1-element.jsを参照


6.idセレクタ
    idセレクタは、指定の要素を検索するためにHTMLタグのid属性を使用して検索する。
    idはページ内でユニークである必要があるので、単一のユニークな要素を検索するために使用する。

    構文
    $("#test")

    例
    chap1-1-element.html/chap1-1-id.jsを参照
